# BMI Calculator

A simple command-line BMI (Body Mass Index) Calculator built using Python.

## Description
This program takes the user's weight (in kilograms) and height (in meters), calculates BMI, and displays the BMI category.

## Features
- User input for weight and height
- Calculates BMI
- Displays BMI rounded to 2 decimal places
- Shows BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese

## BMI Formula
BMI = Weight / (Height × Height)

## Requirements
- Python 3.x

## How to Run

1. Open Python IDLE
2. Create a new file
3. Paste the BMI calculator code
4. Save the file as `bmi.py`
5. Run the program by pressing **F5**
